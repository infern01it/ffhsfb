Linea pro 500

Default

В основе представленного решения лежит система отслеживания товара при помощи RFID – меток, которая позволяет видеть товар на каждом этапе в цепочке «производство — транспортная компания — магазин». С помощью оборудования Linea Pro с опцией чтения RFID – меток осуществляется сканирование товарной единицы, а антенны определяют местоположение товара в режиме реального времени. Визуализация на компьютере отображает фактическое местонахождение любой единицы, тем самым обеспечивая удобство контроля за товаром и его перемещением.

Характеристики

Процессор

  • Процессор 32-bit ARMTM Cortex TM — M

Шифрование и управление ключами (опция)

  • PCI PTS 3.x
  • DUKPT и Fixed Key Management
  • Алгоритмы шифрования 3DES-112, AES-128, AES-256

Чтение RFID меток (опция)

  • MiFARE Mini • ISO 14443
  • MiFARE Classic 1K / 4K • ISO 15693
  • MiFARE Ultralight / Ultralight-C • DESFire card
  • MiFARE Plus 2K / 4K • FeliCa

Чтение штрих-кода(опция)

1D
  • Opticon MDL1000* — Одномерный лазерный сканер штрих-кодов
2D
  • Newland EM3070* — Image Sensor 752×480 CMOS
  • Двухмерный высокопроизводительный сканер штрих-кодов
  • Intermec EA-30 — Image Sensor 752×480 CMOS
  • Двухмерный высокопроизводительный сканер штрих-кодов

Питание

  • 5В через mini-USB
  • 5В через док-станцию

Поддерживаемые типы штрих-кодов

1D Одномерные штрих-коды:
  • JAN/UPC/EAN incl. add on, Codabar/NW-7, Code 11, Code 39, Code 93, Code 128, GS1-128(EAN-128), GS1 DataBar
  • (RSS), IATA, Industrial 2of5, Interleaved 2of5, ISBN-ISMN-ISSN, Matrix 2of5, MSI/Plessey, S-Code, Telepen, Tri-Optic, UK/Plessey
  • Почтовые коды: Chinese Post, Korean Postal Authority code
  • Composite codes, MicroPDF417, PDF417
2D Двухмерные штрих-коды:
  • Code 128, EAN-13, EAN-8, Code 39, UPC-A, UPC-E, Codabar, Interleaved 2 of 5, ITF-6, ITF-14, ISBN, Code 93, UCC/
  • EAN-128, GS1 Databar, Matrix 2 of 5, Code 11, Industrial 2 of 5, Standard 2 of 5, Plessey, MSI-Plessey
  • PDF417, Data Matrix (ECC200, ECC000, 050, 080, 100, 140), QR Code

Батарея

  • Li-Ion аккумулятор 3.7В / 2040мА *(Fuel Gauge option)

Подключение

  • Штекер 9-pin
  • Порт mini-USB
  • Bluetooth 2.0 Class 2 (опция)

Условия

  • Рабочий: -10C до +40C / 5%до 90% RH
  • В режиме покоя: -20C до +50C / 5% до 90% RH
  • Drop Test 1.5 m, 6 faces + 4 edges on concrete

Габариты

Совместимость
